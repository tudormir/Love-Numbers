Love Numbers

You wil see a number appear on the screen. You need to press one of the three button below if the number that you see is an even number, an odd number of a prime number.
  • Even number is a number that can be divided by 2: for example 4.
  • Prime number is a number that can be divided by 1 and by itself: for example 5.
  • Odd number is a number that can not be divided by 2, but not a prime number: for example 9.
